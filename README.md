#### Установка и запуск приложения

<details><summary><b>Развернуть</b></summary>

Клонировать репозиторий:

    git clone https://github.com/Mariyazakharova73/react-dress.git

Установить зависимости:

    npm install

Собрать проект:

    npm run build

Запустить проект:

    npm start

</details>
